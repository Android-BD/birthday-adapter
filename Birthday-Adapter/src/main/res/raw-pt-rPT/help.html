Como é que o Birthday Adapter funciona?

O Birthday Adapter cria um adaptador de sincronização no seu dispositivo e sincroniza todos os eventos como datas de nascimento, aniversários e outros eventos dos seus contactos e mostra-os no calendário Android.

Eu quero sincronizar os eventos com um calendário existente!

Esta não é a ideia desta aplicação. Uma vez que os aniversários dos seus contactos são sincronizados para a nuvem, seria redundante a sincronização de um calendário com a mesma informação. Para esse efeito, utilize as ferramentas do seu computador para mostrar os aniversários dos seus contactos num calendário, tal como o Birthday Adapter faz. Para o Thunderbird, esta operação pode ser efetuada com o extra Thunderbirthday Add-on.

Os eventos não são mostrados no widget de calendário!

O eventos que ocupem todo o dia não são mostrados no dia do evento na versão 4.0 do Android. Este erro foi corrigido na versão 4.0.4. Consulte Android Bug 22365 para mais informações.

Qual a frequência de sincronização dos eventos com o calendário?

A sincronização é diária e automaticamente invocada pelo Android.
