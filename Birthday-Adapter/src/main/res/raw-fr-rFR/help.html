Comment fonctionne Birthday Adapter ?

Birthday Adapter crée un nouveau SyncAdapter sur votre mobile Android, qui synchronise tous les événements, comme les anniversaires, fêtes, etc. depuis votre liste de contacts directement dans un calendrier. Ce calendrier est affiché dans l'Agenda standard d'Android.

Je veux synchroniser les événements dans un calendrier existant !

Ce n'est pas le but de cette application. Vu que vous avez déjà toutes les dates d'anniversaires de vos contacts qui sont synchronisées dans le cloud, ce serait redondant de synchroniser un calendrier avec les mêmes informations. Vous pouvez utiliser des logiciels sur votre PC pour afficher les anniversaire de vos contacts, de la même façon que le fait Birthday Adapter. Par exemple avec Thunderbird, vous pouvez les afficher avec l'add-on Thunderbirthday.

Les événements ne s'affichent pas dans le widget du calendrier Android !

Les événements d'une journée complète ne sont pas affichées le jour-même de l'événement dans Android 4.0. Ce bug est corrigé avec Android 4.04.
⏎Voir Bug Android 22365 pour plus d'informations.

À quelle fréquence sont synchronisés les événements dans le calendrier ?

Elle se fait une fois par jour, et est lancée par Android lui-même avec les autres types de synchronisations.

La synchronisation ne fonctionne pas !

Please check the following:

  • Est-ce que la synchronisation automatique est activée sur votre appareil ? (Vous pouvez l'activer ou la désactiver par application, par exemple avec "Android Power Widget").
